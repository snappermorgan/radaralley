== Description ==

> This Plugin requires [MyMail Newsletter Plugin for WordPress](http://rxa.li/mymail?utm_source=Google+Analytics+for+MyMail)

Integrates Google Analytics with MyMail Newsletter Plugin to track your clicks with the popular Analytics service
Requires the MyMail Newsletter plugin >= 2.0 and the Google Analytics tracking code on the website

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress' built-in Add New Plugin installer
2. Activate the plugin
