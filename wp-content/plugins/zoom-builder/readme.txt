A simple drag & drop interface inside the WordPress admin for building unique layouts for pages with widgets.


== Installation ==

1. Go to WordPress Admin > Plugins > Add New.
2. Select the "Upload" link.
3. Upload the ZIP "zoom-builder.zip" & Install the plugin.
4. Activate ZOOM Builder plugin
